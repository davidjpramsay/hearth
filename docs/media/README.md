# Docs Media

Drop public docs assets here before publishing them to the docs site.

Recommended assets:

- `hearth-demo.mp4`
- `settings-page.png`
- `layouts-page.png`
- `school-page.png`
- `chores-page.png`

For the GitHub Pages docs site, copy the final public assets into:

- `apps/docs/public/media/`

That way they are served from:

- `/media/<filename>`
